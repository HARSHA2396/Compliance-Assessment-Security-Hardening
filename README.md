# Compliance Assessment & Security Hardening Project

## Overview
This project demonstrates hands-on experience in endpoint hardening, policy design, compliance assessment, and cloud security aligned with industry frameworks such as **NIST SP 800-53**, **CMMC**, and enterprise security best practices.

The work simulates a real-world enterprise environment and covers:
- Windows & macOS endpoint hardening with evidence-based validation
- Security policy creation (Email, BYOD)
- Compliance mapping and gap analysis with remediation
- Linux security assessment against CMMC framework
- Cloud server hardening & CASB security analysis

## Skills Demonstrated
- **OS Hardening**: Windows 10/11, macOS, Linux (Ubuntu/CentOS)
- **Security Baselines**: NIST, CIS, DISA STIGs
- **Compliance Frameworks**: NIST SP 800-53 Rev.5, CMMC Level 1/2
- **Risk Assessment**: Vulnerability identification & remediation
- **Security Documentation**: Evidence collection & compliance reporting
- **Cloud Security**: Server hardening, CASB implementation planning

## Tools & Technologies Used
- **OS**: Windows 10/11, macOS, Linux (Ubuntu/CentOS/Kali)
- **Hardening Tools**: Group Policy Editor, Local Security Policy, BitLocker, FileVault
- **Compliance Frameworks**: NIST SP 800-53 Rev.5, CMMC 2.0
- **Documentation**: Markdown, evidence-based screenshots
